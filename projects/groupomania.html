About this project

Groupomania is an internal corporate social network. The user creates his account / log in to access the company's news feed. He can then see the posts of his colleagues and react to them by leaving a comment. He can also create a post that will be editable or deletable. On his profile page the user will be able to consult his information and the list of his posts on the social network. He can also disconnect from the network or delete his account if he wishes.

By carrying out this project I learned a lot from React and its component system but also from NodeJS / Express and all the backend libraries that allowed me to build the API for this project. I also learned the ORM approach to connect my SQL data base to my API.

The project is actually in deployment phase.
