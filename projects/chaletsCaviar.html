About this project

Chalets & Caviar is a wordpress website commissioned by a (fictional) cottage sales agency where they could promote their accommodation. Unfortunately, the project has been lost and is therefore not online. The only remaining documents are visuals of the graphic charter that i made for this project.

Arriving on the website, the user could consult the chalets for sale and for rent, if he was interested in a chalet, the user could contact the agency by filling out a contact form. A system has been set up on the wordpress dashboard to allow publishers and company administrators to easily create / modify / delete product pages.
