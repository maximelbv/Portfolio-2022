About this project

GBAF is an intranet that lists information on banking and financing products for employees of 340 (fictitious) companies. To access it, the employee must create an account by entering several pieces of information. Once logged in, the employee can go to the pages of each actor and leave a comment and a like / dislike. He can also modify his account information in the account page.

By carrying out this project i learned to set up and manage a database and connect it to a website with php to create dynamic functionalities. I also improved my mastery of HTML and CSS languages ​​by integrating the website from a provided wireframe
